Tags de texto

Titulos

sempre usar somente um h1 e para outros titulos usar o h2 ou h3

Negrito

para usar o negrito use a tag b tag b (feita para dar um realse a frase)
Outra opção e a tag strong tag strong(feita para dar enfase a uma frase uma importancia)

linha de divisão

usar a tag hr para fazer uma linha para separar sessões como nesse projeto

Linha pulada

para pular uma linha você deve usar a tag br usada na sessão "Negrito"

Italico

para deixar uma frase em italico (letras deitadas) usar a tag i deitado

Sublinhado

para deixar uma frase sublinhada(traçinho em baixo) usar a tag u traçinho

Letras pequenas

para diminuir o tamanho das letras use a tag small por exemplo
pequeno

Risco

para riscar uma frase use a tag del Riscado

Numero ou frase exponencial

para deixar um numero ou frase exponencial(numero encima ou embaixo) usar a tag sup 3-3
e quando quiser que o numero fique embaixo usar a tag sub 3-3
