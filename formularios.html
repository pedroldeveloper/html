Tags de formulário

Oque é e sua tag type text

os formulários são dados de entrada por exemplo um login ou cadastrado em geral e uma interação entre o seu programa e o usuario.Sua tag e form dentro dela pode estar as tags input que serve para enviar um dado, com o type text, dentro da tag pode colocar tambem o complemento placehoder que faz uma marca falando oque deve ser feito tambem pode colocar o complemento id para dar um nome ao seu campo.Dentro da tag form pode tambem colocar a tag label para inserir uma legenda e o complemento for para relacionar a legenda com o seu input
por exemplo:


Tag type password

coloque para ser um campo de senha para ocultar oque esta sendo escrito com o input com seu type password

Tag type email

coloque para servir como um campo para colocar o email que ele detecta um @ e um . com input e type email

Tag type submit

se usa quando quer enviar depois de preencher os dados com a tag input com o type submit e value para dizer o que e

Campo select

serve para dar multipla escolha para um item que pode ser aberto dentro da tag select com o atributo name tem as tags option colocando value para colocar oque vai ser enviado para o formulario e entre abertura e fechamento e escrito oque vai estar na pagina para impossibilitar de escolher alguma opção dentro do option coloque disabled e selected

Campo Textarea

Tag button

usa a tag button quando for preciso o botão de escolha mas não usar a tag input submit usa a tag button que e bem mais facil por conta de voce somente colocar o texto que ira aparecer para o usuario entre a abertura e o fechamento
