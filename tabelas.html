Tags de tabela

Como criar uma tabela

para criar uma tabela use a tag table dentro a tag tr para representar cada linha dentro da tr a tag th para representar o titulo de uma coluna como uma base(usado somente no primeiro tr) e nos outros tr inserir a tag td para inserir um dado
por exemplo:

Jogo Jogadores Avaliações positivas Avaliações negativas
Blons TD 6 100 75 10
Hollow knight 300 300 0
Terraria 1500 1000 250

Estilização

Voce pode estilizar usando alguns atributos dentro da abertura da tag table como width (para escolher a largura da sua tabela) ou border(para adicionar uma borda a sua tabela)
